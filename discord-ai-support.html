❓ "How do I add the bot to my server?"
AI Response:
To add the bot to your Discord server, follow these steps:
1. Click the invite link provided by your admin
2. Select the server from the dropdown menu
3. Review the permissions and click "Authorize"
4. Complete the captcha if prompted

Need help? Type /help in any channel after adding the bot!
❓ "Why isn't the bot responding?"
AI Response:
If the bot isn't responding, please check:

❓ "Can I customize the welcome message?"
AI Response:
Yes! You can fully customize welcome messages using:
• /welcome set [message] - Set custom welcome text
• /welcome embed - Create rich embed messages
• /welcome variables - See available placeholders like {user}, {server}, etc.

Check out our Knowledge Base article "Custom Welcome Messages" for detailed examples!
