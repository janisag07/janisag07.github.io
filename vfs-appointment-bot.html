🚀 Deployment Guide

Technology Stack

  • Language: Python 3.11+
  • Automation: Selenium / Playwright
  • Notifications: Telegram Bot API, SMTP, Twilio SMS
  • Database: PostgreSQL / SQLite
  • Task Scheduling: Celery + Redis
  • Container: Docker + Docker Compose

📦 Docker Deployment

Build and run with Docker:

# Build the image
docker build -t vfs-appointment-bot .

# Run with environment variables
docker run -d \
--name vfs-bot \
-e TELEGRAM_TOKEN=your_token \
-e SMTP_EMAIL=your_email \
-e CHECK_INTERVAL=5 \
-v $(pwd)/data:/app/data \
vfs-appointment-bot

🐳 Docker Compose

version: '3.8'
services:
bot:
build: .
environment:
- TELEGRAM_TOKEN=${TELEGRAM_TOKEN}
- REDIS_URL=redis://redis:6379
depends_on:
- redis
- postgres
redis:
image: redis:alpine
postgres:
image: postgres:15
environment:
- POSTGRES_DB=vfs_appointments

TELEGRAM_TOKEN=your_telegram_bot_token

🎯 Quick Start

# Clone repository
git clone https://github.com/yourname/vfs-appointment-bot

# Install dependencies
pip install -r requirements.txt

# Configure environment
cp .env.example .env
nano .env

# Run the bot
python main.py
