Recent Projects

💼 CRM ↔ Project Management Sync

Problem: Sales team used HubSpot, dev team used ClickUp. New deals weren't automatically creating project tasks. Manual copying = errors + delays.
Solution: Built two-way Zapier integration. New HubSpot deals → auto-create ClickUp tasks with client info. Task completion → update deal stage in HubSpot.
✨ Result: Zero manual data entry. 100% accuracy. 8 hours/week saved.

🤖 AI-Powered Lead Intake

Problem: Marketing agency received leads via website form, email, and social DMs. Each required different manual processing. Response time: 4-6 hours. No lead qualification.
Solution: Make.com scenario with AI integration. All lead sources → OpenAI API for automatic lead scoring & intent analysis → Airtable database → instant Slack notification with AI summary → auto-send personalized email within 2 minutes.
✨ Result: AI-scored leads. Response time: 2 minutes. 37% increase in qualified leads.

🔄 Two-Way Pipeline Sync

Problem: Client tracked inventory in Google Sheets, customers ordered via WooCommerce. Stock updates were manual and frequently out of sync. Overselling = angry customers.
Solution: Custom REST API integration using Make.com webhooks. WooCommerce order → update Google Sheet stock → if stock < threshold, trigger email alert + pause product listing. Sheet updates → sync to WooCommerce.
✨ Result: Real-time inventory sync. Zero overselling incidents. Customer satisfaction ↑ 42%.
